## 关于 ZeroClaw

### ZeroClaw 是什么？

ZeroClaw 是一个 **AI 助手平台**，可以帮你：

- 💬 **聊天对话** - 和各种 AI 模型交流
- 🔧 **执行任务** - 文件操作、网页搜索、运行代码等
- 📱 **多平台接入** - 通过 Telegram、Discord、微信等使用
- 🤖 **自动化** - 定时执行任务，自动处理消息
- 🔌 **连接硬件** - 控制树莓派、Arduino 等设备

### 为什么选择 ZeroClaw？

| 特点 | 说明 |
|------|------|
| **轻量快速** | 启动只需几毫秒，内存占用小于 5MB |
| **省钱** | 可以运行在 $10 的硬件上 |
| **多模型支持** | 支持 OpenAI、Claude、DeepSeek 等 28+ 模型 |
| **安全** | 本地运行，数据不上传第三方 |
| **可扩展** | 可以自己添加新功能 |

### 适合谁使用？

- **个人用户** - 想要一个私人 AI 助手
- **开发者** - 需要集成 AI 能力到项目中
- **小团队** - 想要团队共享的 AI 助手
- **硬件爱好者** - 想要 AI 控制硬件设备
- **自动化需求** - 需要定时执行任务
